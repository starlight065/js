# Что такое callback и для чего она используется в JavaScript?

**Функция обратного вызова (callback, колбэк)** в JavaScript — это функция, которая передается в другую функцию в качестве аргумента и затем вызывается (выполняется) внутри этой внешней функции.

JavaScript — это однопоточный язык. Он может выполнять только одну задачу одновременно. Если бы функции ждали медленных операций (например, загрузки картинки или запроса к серверу), сайт бы «зависал». Колбэки помогают этого избежать.

1. **Асинхронные операции**
2. **Обработка событий (Event Listening)**

Браузер "слушает" действия пользователя (клик, движение мыши, нажатие клавиши) и вызывает функцию-колбэк только тогда, когда это событие происходит.

1. Методы массивов

**Колбэк** — это старый, базовый способ передачи кода "на потом". Он хорош для простых задач (клик по кнопке, перебор массива), но ужасен для сложных цепочек асинхронных операций.

**Промис** — это современная обертка над асинхронными операциями, которая дарует нам удобный синтаксис, единый стандарт обработки ошибок и железобетонные гарантии выполнения. Именно на промисах построен весь современный асинхронный JavaScript.
