# Почему функции в JavaScript называют "объектами первого класса" (first-class citizens)?

Функции в JavaScript называют **объектами первого класса**, потому что с ними можно работать так же, как с любыми другими значениями.

То есть функцию можно:

* сохранить в переменную
* передать в другую функцию как аргумент
* вернуть из функции
