# Чем стрелочные функции (arrow functions) отличаются от обычных функций?

Стрелочные функции отличаются от обычных прежде всего поведением `this`.

Главное:

* У стрелочных функций нет своего `this`. Они берут его из внешнего контекста.
* У стрелочных функций нет `arguments`.
* Их нельзя использовать как конструктор через `new`.
* У них нет своего `prototype`.

Пример с `this`:

```js
const user = {
  name: 'Ann',
  regular() {
    console.log(this.name);
  },
  arrow: () => {
    console.log(this.name);
  }
};

user.regular(); // Ann
user.arrow(); // обычно undefined
```

`regular` получает `this` во время вызова.\
`arrow` берёт `this` из внешней области, а не из объекта `user`.

Короткий ответ для собеседования:

> Стрелочные функции короче синтаксически, но главное отличие — у них нет собственного `this`, `arguments`, `prototype`, и их нельзя вызывать через `new`. Они удобны, когда нужно сохранить внешний контекст.
