# Что такое функция высшего порядка (higher-order function)?

Функция высшего порядка в JS — это функция, которая либо принимает другую функцию как аргумент (например, `map` или `addEventListener`), либо возвращает функцию как результат (создает замыкание). Это возможно благодаря тому, что в JS функции являются объектами первого класса.
