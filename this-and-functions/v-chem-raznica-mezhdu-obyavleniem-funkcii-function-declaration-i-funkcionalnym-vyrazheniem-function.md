# В чем разница между объявлением функции (function declaration) и функциональным выражением (function

Разница — в способе создания функции и в поведении до момента объявления.

Главное:

* **Function declaration** создаётся как отдельное объявление функции.
* **Function expression** — это функция внутри выражения, обычно присваивается переменной.
* **Function declaration** можно вызвать до строки, где она объявлена.
* **Function expression** до присваивания вызвать нельзя.

Пример:

```js
sayHi(); // работает

function sayHi() {
  console.log('Hi');
}
```

```js
sayHello(); // ошибка

const sayHello = function () {
  console.log('Hello');
};
```

Почему так:

* `function declaration` полностью поднимается вверх.
* У `function expression` поднимается только переменная.
* Сама функция становится доступна только после присваивания.

Короткий ответ для собеседования:

> `Function declaration` — это обычное объявление функции, и его можно вызвать до места объявления из-за hoisting. `Function expression` — это функция внутри выражения, обычно в переменной, и вызвать её до присваивания нельзя.
