# Что такое ключевое слово this и как определяется его значение?

`this` — это ссылка на объект, в контексте которого выполняется функция.

Значение `this` определяется не местом объявления функции, а способом её вызова.

* Обычный вызов функции — `this` обычно `undefined` в strict mode.
* Вызов как метода объекта — `this` равен этому объекту.
* Через `call`, `apply`, `bind` — `this` задаётся явно.
* В стрелочных функциях своего `this` нет. Они берут его из внешнего контекста.

Пример:

```js
const user = {
  name: 'Ann',
  sayHi() {
    console.log(this.name);
  }
};

user.sayHi(); // Ann
```

Короткий ответ для собеседования:

> `this` в JavaScript — это объект текущего вызова функции. Его значение определяется тем, как функция вызвана, а не тем, где она объявлена. У стрелочных функций `this` берётся из внешнего контекста.
