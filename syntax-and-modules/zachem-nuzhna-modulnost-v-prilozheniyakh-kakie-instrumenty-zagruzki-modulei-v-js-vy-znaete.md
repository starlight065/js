# Зачем нужна модульность в приложениях? Какие инструменты загрузки модулей в JS вы знаете

Раньше для серверов использовали CommonJS, для браузеров — AMD[^1], а для универсальности — UMD[^2]. Сегодня индустрия полностью перешла на официальный стандарт ES-модули (ESM), который работает везде.





[^1]: Asynchronous Module Definition

[^2]: Universal Module Definition
