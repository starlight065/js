# Порядок выполнения синхронного кода, промисов (Promises) и setTimeout()

Порядок выполнения: синхронный код → микрозадачи → макрозадачи

**Приоритет:**

1. Синхронный код (Call Stack) — выполняется первым, целиком
2. Микрозадачи (`Promise.then`/`catch`/`finally`, `queueMicrotask`) — выполняются **все**, пока очередь не опустеет
3. Макрозадачи (`setTimeout`, `setInterval`) — выполняется **по одной**, затем снова проверяются микрозадачи

**Правило:** после каждой макрозадачи event loop сначала опустошает очередь микрозадач, и только потом берёт следующую макрозадачу.
