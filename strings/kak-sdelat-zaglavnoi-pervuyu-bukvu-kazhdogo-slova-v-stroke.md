# Как сделать заглавной первую букву каждого слова в строке?

```javascript
function capitalizeWordsRegex(str) {
    // Сначала приводим к нижнему регистру, а затем заменяем первые буквы на заглавные
    return str.toLowerCase().replace(/(^|\s)\S/g, l => l.toUpperCase());
}

console.log(capitalizeWordsRegex("быстрый   бурый лис")); 
// Вывод: "Быстрый   Бурый Лис" (лишние пробелы сохранились, а буквы стали заглавными)
```

Мы ищем первый символ строки (`^`) и любой символ, перед которым стоит пробел (`\s`), и заменяем его на заглавный.

* `(^|\s)` — находит начало строки ИЛИ пробел.
* `\S` — находит любой непробельный символ (т.е. первую букву слова).
* `g` — флаг «global» (искать по всей строке).
* `l => l.toUpperCase()` — функция, которая делает найденный символ заглавным.
