# Пробелы, удаление символов

В JavaScript для удаления пробелов и дополнения строк существуют удобные встроенные методы. Давайте разберем обе группы методов подробно.

***

### Часть 1. Удаление пробелов в начале и конце строки

Все эти методы **не изменяют исходную строку**, а возвращают новую, так как строки в JavaScript неизменяемы (immutable). Они удаляют не только обычные пробелы, но и символы табуляции (`\t`), перевода строки (`\n`) и другие невидимые символы.

#### 1. `trim()` — удаляет пробелы с обоих концов строки

Самый часто используемый метод. Он очищает строку и слева, и справа.

#### 2. `trimStart()` (или `trimLeft()`) — удаляет пробелы только в начале

Удаляет пробелы только с левой стороны строки.

#### 3. `trimEnd()` (или `trimRight()`) — удаляет пробелы только в конце

Удаляет пробелы только с правой стороны строки.

***

### Часть 2. Дополнение строки до заданной длины

Методы `padStart` и `padEnd` позволяют дополнить текущую строку другой строкой (или символом) до тех пор, пока итоговая строка не достигнет указанной длины.

**Синтаксис:** `str.padStart(targetLength, padString)`

* `targetLength` — желаемая длина итоговой строки.
* `padString` (необязательный) — символ(ы), которыми будет дополняться строка. По умолчанию это обычный пробел `" "`.

***

#### 1. `padStart()` — дополняет строку в НАЧАЛЕ

Часто используется для форматирования чисел (например, дат или времени), чтобы они всегда имели одинаковую длину.

**Пример 1: Форматирование времени (добавление ведущего нуля)**

```javascript
const minutes = "9";
const formattedMinutes = minutes.padStart(2, "0");

console.log(formattedMinutes); // "09" (длина строки стала 2)
```

**Пример 2: Маскирование номера карты**

```javascript
const lastFourDigits = "4321";
// Дополняем звездочками до общей длины 16
const maskedCard = lastFourDigits.padStart(16, "*");

console.log(maskedCard); // "************4321"
```

***

#### 2. `padEnd()` — дополняет строку в КОНЦЕ

Добавляет символы с правой стороны строки.

**Пример 1: Создание красивых строк/чеков**

```javascript
const item = "Хлеб";
const price = "50р";

// Дополняем название товара точками до 20 символов в конце
const line = item.padEnd(20, ".") + price;

console.log(line); // "Хлеб................50р"
```
