# Что такое промис (Promise) и какие три состояния у него выделяют?

**Promise** — объект, представляющий результат асинхронной операции, который будет доступен в будущем (сейчас, позже или никогда).

Три состояния:

1. **pending** — начальное состояние, результат ещё не готов
2. **fulfilled** — операция завершилась успешно, есть результат
3. **rejected** — операция завершилась с ошибкой

Особенности:

* Промис может перейти из pending только в fulfilled или rejected — назад пути нет
* После смены состояния оно больше не меняется (immutable)
* `.then()` — обработка успеха, `.catch()` — обработка ошибки, `.finally()` — выполняется в любом случае
