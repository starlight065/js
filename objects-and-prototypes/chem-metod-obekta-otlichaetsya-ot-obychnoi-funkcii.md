# Чем метод объекта отличается от обычной функции?

**Метод** — это функция, которая хранится в свойстве объекта.

```js
const user = {
  sayHi() {
    console.log('Hi');
  }
};
```

Технически метод — это тоже функция. Разница в том, **как она используется**.

* метод вызывают через объект;
* у метода `this` обычно указывает на этот объект;
* обычная функция не привязана к объекту сама по себе.

```js
const user = {
  name: 'Ann',
  sayHi() {
    console.log(this.name);
  }
};

user.sayHi(); // Ann
```

Короткий ответ для собеседования:

> Метод объекта — это функция, записанная в свойство объекта. Главное отличие в контексте вызова. При вызове как `obj.method()` значение `this` обычно равно `obj`.
