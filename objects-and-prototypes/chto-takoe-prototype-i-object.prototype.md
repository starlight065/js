# Что такое \[\[Prototype]] и Object.prototype?

`[[Prototype]]` — это внутреннее скрытое свойство объекта. Оно указывает на другой объект, у которого можно искать свойства.

`Object.prototype` — это базовый объект, от которого в итоге наследуется большинство обычных объектов.

```js
const user = { name: 'Ann' };

Object.getPrototypeOf(user) === Object.prototype; // true
```

Когда вы обращаетесь к свойству, JavaScript сначала ищет его в самом объекте, потом в `[[Prototype]]`, потом выше по цепочке.

Короткий ответ для собеседования:

> `[[Prototype]]` — это скрытая ссылка на прототип объекта. `Object.prototype` — верхний базовый прототип для большинства объектов. Через эту связь работает наследование и поиск свойств.
