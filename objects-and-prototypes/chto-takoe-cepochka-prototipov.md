# Что такое цепочка прототипов?

Если у объекта нет нужного свойства, JavaScript ищет его в прототипе. Если там тоже нет, поиск идёт дальше по цепочке.

Это и есть **цепочка прототипов**.

```js
const animal = {
  eats: true
};

const dog = Object.create(animal);
console.log(dog.eats); // true
```

В конце цепочки обычно стоит `Object.prototype`, а потом `null`.

Короткий ответ для собеседования:

> Цепочка прототипов — это механизм поиска свойств. Если свойства нет в самом объекте, JavaScript поднимается по его прототипам, пока не найдёт значение или не дойдёт до `null`.
