# Как проверить, что объект создан этим классом?

Самый частый способ — оператор `instanceof`.

```js
class User {}
const user = new User();

console.log(user instanceof User); // true
```

Он проверяет, есть ли `User.prototype` в цепочке прототипов объекта.

Дополнительно можно использовать:

* `obj.constructor === User` — менее надёжно;
* `User.prototype.isPrototypeOf(obj)` — тоже работает.

Короткий ответ для собеседования:

> Обычно используют `instanceof`. Этот оператор проверяет не имя класса, а наличие его прототипа в цепочке прототипов объекта.
