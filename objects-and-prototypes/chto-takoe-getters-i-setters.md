# Что такое getters и setters?

**Getter** — это метод, который срабатывает при чтении свойства.

**Setter** — это метод, который срабатывает при записи свойства.

```js
const user = {
  firstName: 'Ann',
  lastName: 'Lee',

  get fullName() {
    return `${this.firstName} ${this.lastName}`;
  },

  set fullName(value) {
    [this.firstName, this.lastName] = value.split(' ');
  }
};
```

Они нужны, когда:

* надо скрыть внутреннюю логику;
* надо валидировать данные при записи;
* надо вычислять значение при чтении.

Короткий ответ для собеседования:

> Getter и setter позволяют читать и изменять свойство через методы, но выглядят как обычное свойство. Getter используют для вычисления значения, setter — для контроля и валидации при записи.
