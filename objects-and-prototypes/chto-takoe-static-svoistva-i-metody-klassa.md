# Что такое static свойства и методы класса?

`static` свойства и методы принадлежат **самому классу**, а не его экземплярам.

```js
class User {
  static role = 'user';

  static createGuest() {
    return new User('Guest');
  }

  constructor(name) {
    this.name = name;
  }
}

console.log(User.role); // user
```

Их используют, когда логика относится ко всему классу:

* фабричные методы;
* константы;
* вспомогательные утилиты.

Короткий ответ для собеседования:

> Static свойства и методы вызывают на классе, а не на экземпляре. Они нужны для общей логики, которая не зависит от конкретного объекта.
