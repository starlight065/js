# Как проверить наличие свойства у объекта?

Есть три основных способа.

* `key in obj` — проверяет и собственные, и наследуемые свойства;
* `Object.hasOwn(obj, key)` — проверяет только собственные свойства;
* `obj.hasOwnProperty(key)` — старый, но всё ещё рабочий вариант.

```js
const user = { name: 'Ann' };

'name' in user; // true
Object.hasOwn(user, 'name'); // true
```

Если на собеседовании спрашивают про надёжный современный вариант, лучше назвать `Object.hasOwn()`.

Короткий ответ для собеседования:

> Для проверки свойства можно использовать `in`, `Object.hasOwn()` и `hasOwnProperty()`. `in` видит и свойства из прототипа, а `Object.hasOwn()` проверяет только собственные свойства объекта.
