# Что такое объект и как получить доступ к его свойствам?

**Объект** — это структура данных, которая хранит значения по ключам.

```js
const user = {
  name: 'Ann',
  age: 25
};
```

Получить доступ к свойствам можно двумя способами:

* через точку — `user.name`;
* через квадратные скобки — `user['name']`.

Скобки нужны, когда:

* имя свойства хранится в переменной;
* в имени есть пробелы или спецсимволы.

```js
const key = 'age';
console.log(user[key]); // 25
```

Короткий ответ для собеседования:

> Объект в JavaScript — это набор пар ключ-значение. К свойствам обращаются через точку или квадратные скобки. Скобки используют для динамических ключей.
