# Как получить все ключи и значения объекта?

Для этого используют стандартные методы объекта `Object`.

* `Object.keys(obj)` — массив ключей;
* `Object.values(obj)` — массив значений;
* `Object.entries(obj)` — массив пар `[key, value]`.

```js
const user = { name: 'Ann', age: 25 };

Object.keys(user);   // ['name', 'age']
Object.values(user); // ['Ann', 25]
Object.entries(user); // [['name', 'Ann'], ['age', 25]]
```

Если нужны только **собственные** свойства, это лучший вариант. Наследуемые свойства эти методы не возвращают.

Короткий ответ для собеседования:

> Для получения ключей, значений и пар ключ-значение используют `Object.keys`, `Object.values` и `Object.entries`. Эти методы работают только с собственными перечисляемыми свойствами объекта.
