# Как удалить свойство из объекта?

Для удаления свойства используют оператор `delete`.

```js
const user = { name: 'Ann', age: 25 };
delete user.age;

console.log(user); // { name: 'Ann' }
```

Важно помнить:

* `delete` удаляет свойство объекта;
* он не удаляет локальные переменные;
* если свойства нет, ошибки не будет.

В массивах `delete arr[i]` использовать не стоит. Он оставляет пустую ячейку. Для массивов лучше `splice()`.

Короткий ответ для собеседования:

> Свойство объекта удаляют через `delete obj.key`. Для объектов это нормально. Для массивов обычно используют не `delete`, а `splice`, чтобы не оставлять пустые элементы.
