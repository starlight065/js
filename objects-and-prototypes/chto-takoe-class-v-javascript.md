# Что такое class в JavaScript?

`class` — это удобный синтаксис для создания объектов и наследования поверх прототипов.

```js
class User {
  constructor(name) {
    this.name = name;
  }

  sayHi() {
    console.log(this.name);
  }
}
```

Важно понимать:

* `class` не отменяет прототипы;
* методы класса хранятся в `prototype`;
* `new User()` создаёт экземпляр объекта.

Короткий ответ для собеседования:

> `class` в JavaScript — это синтаксический сахар над прототипным наследованием. Он делает код ближе к ООП-стилю, но под капотом всё равно работает через функции и прототипы.
