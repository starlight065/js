# В чем разница между оператором spread и rest?

Оба используют синтаксис `...`, но работают по-разному.

`spread` разворачивает массив, объект или аргументы в отдельные элементы. Его используют в массивах, объектах и при вызове функций.

```js
const nums = [1, 2, 3];
const copy = [...nums];
```

`rest` собирает данные в один массив. Его часто используют в параметрах функции или деструктуризации.

```js
function sum(...args) {
  return args.reduce((acc, n) => acc + n, 0);
}
```

Как запомнить:

* `spread` — разложить;
* `rest` — собрать остаток.

Короткий ответ для собеседования:

> `spread` разворачивает массив, объект или список аргументов. `rest` наоборот собирает несколько значений в одну структуру. Синтаксис одинаковый, но смысл зависит от места использования.
