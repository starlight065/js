# Объясните деструктуризацию для объектов и массивов

Деструктуризация позволяет быстро достать значения из массива или объекта в отдельные переменные.

Для массива:

```js
const arr = [10, 20, 30];
const [a, b] = arr;
```

Здесь `a` будет `10`, а `b` будет `20`.

Для объекта:

```js
const user = { name: 'Ann', age: 25 };
const { name, age } = user;
```

Это полезно, когда:

* нужно короче писать код;
* нужно быстро взять нужные поля;
* нужно удобно работать с параметрами функции.

Короткий ответ для собеседования:

> Деструктуризация — это удобный способ распаковать значения из массива или свойства из объекта в переменные. Она делает код короче и особенно полезна при работе с объектами, массивами и параметрами функций.
