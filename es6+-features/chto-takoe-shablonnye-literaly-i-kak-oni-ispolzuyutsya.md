# Что такое шаблонные литералы и как они используются?

Шаблонные литералы — это строки в обратных кавычках: `` ` ``.

Они удобны в трёх случаях:

* можно вставлять переменные через `${...}`;
* можно писать многострочные строки;
* можно делать строку более читаемой.

Пример:

```js
const name = 'Ann';
const age = 25;

const text = `Имя: ${name}, возраст: ${age}`;
```

Это удобнее, чем конкатенация через `+`.

Короткий ответ для собеседования:

> Шаблонные литералы — это строки в обратных кавычках. Они позволяют вставлять выражения через `${...}` и удобно писать многострочный текст. Чаще всего их используют вместо склейки строк через `+`.
