# Какие основные методы массивов в JavaScript?

У массивов есть несколько основных групп методов.

Чаще всего на собеседовании ждут такую классификацию:

* **Добавление и удаление** — `push`, `pop`, `shift`, `unshift`, `splice`.
* **Преобразование** — `map`, `filter`, `reduce`.
* **Поиск и проверка** — `find`, `findIndex`, `includes`, `some`, `every`.
* **Копирование и извлечение** — `slice`, `concat`, `flat`.
* **Сортировка** — `sort`, `reverse`, `toSorted`.

Полезно сразу делить методы на две группы:

* **мутирующие** — меняют исходный массив;
* **немутирующие** — возвращают новый массив или новое значение.

Пример:

```js
const numbers = [1, 2, 3, 4];

const doubled = numbers.map(n => n * 2);
const even = numbers.filter(n => n % 2 === 0);
const firstBig = numbers.find(n => n > 2);
```

Короткий ответ для собеседования:

> Основные методы массивов обычно делят на методы изменения, преобразования, поиска и сортировки. Самые важные — `push`, `pop`, `splice`, `slice`, `map`, `filter`, `reduce`, `find`, `some`, `every` и `sort`. Важно понимать, какие из них мутируют исходный массив, а какие нет.
