# Как преобразовать вложенный массив в плоский (flatten)?

В современных проектах для этого обычно используют `flat()`.

```js
const arr = [1, [2, 3], [4, [5]]];

console.log(arr.flat()); // [1, 2, 3, 4, [5]]
console.log(arr.flat(Infinity)); // [1, 2, 3, 4, 5]
```

Что важно знать:

* `flat()` не мутирует исходный массив;
* глубину можно передать числом;
* `flat(Infinity)` разворачивает массив полностью.

Если нужно сначала преобразовать элементы, а потом сделать один уровень плоским, используют `flatMap()`.

```js
const words = ['hi', 'js'];
console.log(words.flatMap(word => word.split('')));
```

Если `flat()` недоступен, можно сказать, что задачу решают через рекурсию или `reduce()`.

Это полезно на собеседовании, если спрашивают не только API, но и понимание алгоритма.

Короткий ответ для собеседования:

> Для выравнивания вложенного массива обычно используют `flat()`. Один уровень раскрывает `flat()`, все уровни — `flat(Infinity)`. Если нужна совместимость со старой средой, задачу можно решить через рекурсию или `reduce()`.
