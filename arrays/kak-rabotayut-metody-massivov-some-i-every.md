# Как работают методы массивов some() и every()?

Оба метода проверяют массив по условию.

Разница такая:

* `some()` возвращает `true`, если условие выполняется **хотя бы для одного** элемента;
* `every()` возвращает `true`, если условие выполняется **для всех** элементов.

```js
const numbers = [2, 4, 6];

numbers.some(n => n > 5);  // true
numbers.every(n => n > 1); // true
numbers.every(n => n > 3); // false
```

Что важно сказать:

* оба метода возвращают `boolean`;
* оба не мутируют массив;
* оба могут завершиться раньше, если результат уже понятен.

Это делает их удобными для проверок и валидации.

Короткий ответ для собеседования:

> `some()` проверяет, есть ли в массиве хотя бы один подходящий элемент. `every()` проверяет, подходят ли все элементы. Оба метода возвращают `boolean` и часто используются для условий и валидации.
