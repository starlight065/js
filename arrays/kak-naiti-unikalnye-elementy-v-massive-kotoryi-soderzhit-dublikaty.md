# Как найти уникальные элементы в массиве, который содержит дубликаты?

Самый простой способ — использовать `Set`.

`Set` хранит только уникальные значения.

```js
const arr = [1, 2, 2, 3, 3, 4];
const unique = [...new Set(arr)];

console.log(unique); // [1, 2, 3, 4]
```

Почему это хороший ответ:

* код короткий;
* читается сразу;
* сохраняет порядок первого появления элементов.

Если массив содержит объекты, `Set` сравнивает их по ссылке. Тогда уникальность часто ищут по полю через `Map` или `reduce()`.

```js
const users = [
  { id: 1, name: 'Ann' },
  { id: 1, name: 'Ann 2' },
  { id: 2, name: 'Bob' }
];

const uniqueUsers = [...new Map(users.map(user => [user.id, user])).values()];
```

Короткий ответ для собеседования:

> Для массива примитивов проще всего использовать `Set`: `[...new Set(arr)]`. Это убирает дубликаты и сохраняет порядок. Для массива объектов уникальность обычно определяют по ключу, например через `Map`.
