# Что такое массив? Как можно перебрать (итерировать) элементы массива?

Массив — это упорядоченная коллекция значений.

Элементы массива имеют числовые индексы, начиная с `0`.

Перебирать массив можно несколькими способами:

* `for` — когда нужен полный контроль над индексом;
* `for...of` — когда важны сами элементы;
* `forEach()` — когда нужно выполнить действие для каждого элемента.

```js
const arr = ['a', 'b', 'c'];

for (const item of arr) {
  console.log(item);
}

arr.forEach((item, index) => {
  console.log(index, item);
});
```

Что полезно сказать на собеседовании:

* `for...in` для массивов обычно не используют;
* `map()`, `filter()` и `reduce()` тоже проходят по массиву, но они больше про преобразование данных;
* выбор способа зависит от задачи.

Короткий ответ для собеседования:

> Массив — это структура данных для хранения упорядоченного набора значений. Перебирать элементы можно через `for`, `for...of` и `forEach()`. Для массивов чаще всего выбирают `for...of` или методы массива, а `for...in` стараются избегать.
