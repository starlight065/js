# Объясни разницу между методами splice и slice.

Разница очень важная:

* `splice()` **меняет** исходный массив;
* `slice()` **не меняет** исходный массив.

`splice()` используют, когда нужно удалить, заменить или вставить элементы по индексу.

```js
const arr = [1, 2, 3, 4];
arr.splice(1, 2, 'a');

console.log(arr); // [1, 'a', 4]
```

`slice()` используют, когда нужно получить часть массива.

```js
const arr = [1, 2, 3, 4];
const part = arr.slice(1, 3);

console.log(part); // [2, 3]
console.log(arr);  // [1, 2, 3, 4]
```

Полезно помнить и по сигнатуре:

* `slice(start, end)` — возвращает копию диапазона;
* `splice(start, deleteCount, ...items)` — удаляет и вставляет элементы.

Запомнить можно так:

* `splice` — surgery, вмешивается в массив;
* `slice` — срез, только читает часть.

Короткий ответ для собеседования:

> `splice()` мутирует массив и умеет удалять, вставлять и заменять элементы. `slice()` не мутирует массив и возвращает его часть. На практике `splice()` используют для изменения массива, а `slice()` — для безопасного получения копии или диапазона элементов.
