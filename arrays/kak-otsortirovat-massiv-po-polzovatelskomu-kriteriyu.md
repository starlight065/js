# Как отсортировать массив по пользовательскому критерию?

Для этого в `sort()` передают функцию-компаратор.

Компаратор получает два элемента и должен вернуть:

* число меньше `0`, если `a` должно идти раньше `b`;
* число больше `0`, если `b` должно идти раньше `a`;
* `0`, если порядок не важен.

```js
const users = [
  { name: 'Bob', age: 30 },
  { name: 'Ann', age: 25 }
];

users.sort((a, b) => a.age - b.age);
```

Можно сортировать по числу, строке или длине поля.

```js
users.sort((a, b) => a.name.localeCompare(b.name));
```

Важно помнить, что `sort()` мутирует исходный массив. Если нужен новый массив, лучше использовать `toSorted()`.

Короткий ответ для собеседования:

> Чтобы отсортировать массив по своему правилу, в `sort()` передают компаратор. Например, для чисел это часто `(a, b) => a - b`, а для строк — `localeCompare()`. Нужно помнить, что `sort()` меняет исходный массив.
