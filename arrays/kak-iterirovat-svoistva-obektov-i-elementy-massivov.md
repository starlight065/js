# Как итерировать свойства объектов и элементы массивов?

Для массивов и объектов обычно используют разные подходы.

Для массива подходят:

* `for...of`
* `forEach()`
* обычный `for`
* `entries()` для индекса и значения

```js
const arr = ['a', 'b', 'c'];

for (const value of arr) {
  console.log(value);
}
```

Для объекта чаще используют `Object.keys()`, `Object.values()` и `Object.entries()`.

```js
const user = { name: 'Ann', age: 25 };

for (const [key, value] of Object.entries(user)) {
  console.log(key, value);
}
```

Важно:

* `for...in` перебирает перечислимые ключи объекта.
* Для массивов `for...in` обычно не рекомендуют.
* Для массива лучше `for...of` или методы массива.

Короткий ответ для собеседования:

> Элементы массива обычно перебирают через `for...of`, `forEach()` или обычный `for`. Свойства объекта обычно перебирают через `Object.keys()`, `Object.values()` или `Object.entries()`. `for...in` больше подходит для объектов и обычно не используется для массивов.
