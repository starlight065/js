# Какие методы для поиска в массиве вы знаете?

Для поиска в массиве чаще всего используют несколько методов.

Основные:

* `find()` — возвращает первый подходящий элемент;
* `findIndex()` — возвращает индекс первого подходящего элемента;
* `includes()` — проверяет наличие значения;
* `indexOf()` — ищет индекс значения;
* `some()` — проверяет, есть ли хотя бы один подходящий элемент.

```js
const numbers = [5, 10, 15];

numbers.find(n => n > 8);      // 10
numbers.findIndex(n => n > 8); // 1
numbers.includes(10);          // true
numbers.indexOf(15);           // 2
numbers.some(n => n > 20);     // false
```

Что важно помнить:

* `find()` возвращает сам элемент;
* `findIndex()` возвращает индекс или `-1`;
* `includes()` удобен, когда нужно просто проверить наличие значения.

Короткий ответ для собеседования:

> Для поиска в массиве обычно используют `find()`, `findIndex()`, `includes()`, `indexOf()` и `some()`. Выбор зависит от того, что нужно получить: элемент, индекс или просто факт наличия.
