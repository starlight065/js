# Что такое JSON и как работают методы JSON.parse() и JSON.stringify()?

**JSON** (JavaScript Object Notation) — это популярный текстовый формат обмена данными. Он используется для передачи данных между клиентом (браузером) и сервером, а также для хранения конфигурационных файлов.

Несмотря на то, что JSON изначально произошел из JavaScript, он является **языконезависимым**. Практически все современные языки программирования умеют работать с JSON.

```jsx
JSON.stringify(value, replacer, space)
```

Этот метод берет JavaScript-объект (или массив/значение) и превращает его в **строку** в формате JSON. Это нужно, например, чтобы отправить данные на сервер или сохранить их в `localStorage`.

```jsx
JSON.parse(text, reviver)
```

Этот метод делает обратную операцию: берет **строку** в формате JSON и превращает её обратно в **JavaScript-объект** (или массив).

> С помощью связки этих двух методов часто делают глубокое копирование простых объектов
