# Какие виды циклов в JavaScript вы знаете? В каких случаях лучше использовать каждый из них?

В JavaScript циклы используются для многократного повторения определенных участков кода. Давайте подробно разберем четыре основных типа циклов: `for`, `while`, `for...of` и `for...in`.

***

#### 1. Классический цикл `for`

Используется, когда вы **заранее знаете, сколько раз** должен выполниться цикл (например, нужно повторить действие ровно 10 раз или пройти по массиву по его индексам).

**Синтаксис:**

* **Инициализация:** выполняется один раз перед началом цикла. Обычно здесь создается переменная-счетчик (например, `let i = 0`).
* **Условие:** проверяется перед каждой итерацией. Если оно истинно (`true`), цикл продолжается; если ложно (`false`), цикл завершается.
* **Шаг:** выполняется после каждой итерации (например, увеличение счетчика `i++`).

**Пример:**

```jsx
for (let i = 0; i < 5; i++) {
  console.log(`Итерация: ${i}`);
}
// Выведет числа от 0 до 4
```

***

#### 2. Цикл `while`

Используется, когда количество итераций **заранее неизвестно**, и цикл должен выполняться до тех пор, пока истинно определенное условие.

**Синтаксис:**

```jsx
while (условие) {
  // код цикла
}
```

**Пример:**

```jsx
let count = 0;

while (count < 3) {
  console.log(`Счетчик: ${count}`);
  count++; // Важно изменять условие, иначе цикл станет бесконечным!
}
```

> **Родственник: цикл `do...while`**\
> Отличается тем, что условие проверяется _после_ выполнения тела цикла. Это гарантирует, что код выполнится **хотя бы один раз**.
>
> ```jsx
> let x = 5;
> do {
>   console.log("Этот код выполнится 1 раз, даже если условие ложно");
> } while (x < 3);
> ```

***

#### 3. Цикл `for...of` (для значений)

Появился в ES6. Он используется для перебора **значений** в итерируемых объектах (массивы, строки, коллекции Map и Set).

* Он **не работает** с обычными объектами (Object), так как они не являются итерируемыми.
* В нем нельзя напрямую получить индекс элемента (хотя можно использовать `array.entries()`).

**Пример с массивом:**

```jsx
const fruits = ['яблоко', 'банан', 'апельсин'];

for (const fruit of fruits) {
  console.log(fruit);
}
// Выведет: 'яблоко', 'банан', 'апельсин'
```

**Пример со строкой:**

```jsx
const word = "JS";
for (const char of word) {
  console.log(char);
}
// Выведет: "J", "S"
```

***

#### 4. Цикл `for...in` (для свойств/ключей)

Используется для перебора всех перечисляемых свойств **объекта** (его **ключей** или названий свойств).

* Можно использовать и для массивов (он вернет индексы как строки), но делать это **крайне не рекомендуется**, так как он работает медленнее и может обходить свойства, добавленные в прототип массива.

**Синтаксис:**

```jsx
for (const ключ in объект) {
  // код
}
```

**Пример с объектом:**

```jsx
const user = {
  name: 'Алексей',
  age: 30,
  role: 'admin'
};

for (const key in user) {
  console.log(`Ключ: ${key}, Значение: ${user[key]}`);
}
/* Выведет:
   Ключ: name, Значение: Алексей
   Ключ: age, Значение: 30
   Ключ: role, Значение: admin
*/
```

***

#### Управление циклами: `break` и `continue`

Во всех этих циклах можно управлять процессом выполнения:

* **`break`** — полностью прерывает выполнение цикла и выходит из него.
* **`continue`** — прерывает текущую итерацию и сразу переходит к следующей.

**Пример:**

```jsx
for (let i = 0; i < 10; i++) {
  if (i === 3) continue; // пропускаем число 3
  if (i === 6) break;    // останавливаем цикл на 6
  console.log(i);
}
// Выведет: 0, 1, 2, 4, 5
```
