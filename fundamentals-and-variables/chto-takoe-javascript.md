# Что такое JavaScript

**JavaScript (JS)** — это мультипарадигменный динамический язык программирования, который является «языком веб-браузеров» и стандартом для создания интерактивных интерфейсов.

**Для чего он используется сегодня:**

1. **Frontend (клиентская часть):** Это его главная роль. JS «оживляет» веб-страницы (отвечает за анимацию, обработку кликов, отправку форм без перезагрузки страницы) и используется для создания сложных веб-приложений (на фреймворках React, Vue, Angular).
2. **Backend (серверная часть):** Благодаря платформе **Node.js** на JS пишут серверную логику, API и работают с базами данных.
3. **Мобильная и десктопная разработка:** С помощью фреймворков (React Native, Electron) на JS создают кроссплатформенные приложения (например, Discord, Skype, VS Code).

Это единственный язык, который позволяет разработчику быть **Fullstack-специалистом** (писать и клиентскую, и серверную часть приложения), используя один стек технологий.
