# Что такое рекурсия (recursion)? Что такое стек (stack)?

**Рекурсия** — это прием, когда функция **вызывает саму себя** внутри своего тела.

Любая правильная рекурсивная функция должна состоять из двух частей:

1. **Базовый случай (условие выхода).** Точка, в которой рекурсия останавливается;
2. Рекурсивный шаг (вызов функцией самой себя, обычно с измененными параметрами)

```jsx
function factorial(n) {
  // 1. Базовый случай (выход из рекурсии)
  if (n === 1) {
    return 1;
  }
  // 2. Рекурсивный шаг
  return n * factorial(n - 1);
}

console.log(factorial(3)); // Выведет 6
```

* Стек — структура данных, которая работает по принципу LIFO (Last In, First Out);
* Стек вызовов (Call Stack) — механизм, с помощью которого JavaScript следит за тем, какая функция выполняется в данный момент, и какие функции вызываются внутри неё
* Когда функция **вызывается**, она попадает на **вершину стека**.
* Когда функция **завершает выполнение** (встречает `return` или закрывающую скобку), она **удаляется из стека**.
