# Какие логические операторы есть в JavaScript и какой результат они возвращают?

**`|| (OR)`** возвращает первое truthy (истинное) значение

&#x20;**`&& (AND)`** — первое falsy (ложное) значение

**`??`** — первое определенное значение (не `null` и не `undefined`).

Исключением является оператор **`! (NOT)`**, который всегда приводит операнд к булевому типу (`true/false`) и инвертирует его.
