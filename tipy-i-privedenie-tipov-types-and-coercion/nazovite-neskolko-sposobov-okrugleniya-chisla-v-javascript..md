# Назовите несколько способов округления числа в JavaScript.

#### 1. `Math.round(x)` — Округление по математическим правилам

Округляет число к ближайшему целому. Если дробная часть равна `0.5` или больше, число округляется в большую сторону (к +∞).

```jsx
Math.round(4.4);  // 4
Math.round(4.5);  // 5
Math.round(4.6);  // 5

// Внимание с отрицательными числами:
Math.round(-4.5); // -4 (так как -4 больше, чем -5)
Math.round(-4.51);// -5
```

#### 2. `Math.floor(x)` — Округление вниз («в пол»)

Всегда округляет число к **меньшему** целому (в сторону отрицательной бесконечности).

```jsx
Math.floor(4.9);   // 4
Math.floor(4.1);   // 4

// Внимание с отрицательными числами:
Math.floor(-4.1);  // -5 (так как -5 меньше, чем -4)
```

#### 3. `Math.ceil(x)` — Округление вверх («в потолок»)

Всегда округляет число к **большему** целому (в сторону положительной бесконечности).

```jsx
Math.ceil(4.1);   // 5
Math.ceil(4.9);   // 5

// Внимание с отрицательными числами:
Math.ceil(-4.9);  // -4 (так как -4 больше, чем -5)
```

#### 4. `Math.trunc(x)` — Усечение (отбрасывание дробной части)

Просто удаляет все цифры после запятой, не производя никакого округления. Направление зависит от знака: для положительных чисел работает как `floor`, для отрицательных — как `ceil` (округляет в сторону нуля).

```jsx
Math.trunc(4.9);   // 4
Math.trunc(-4.9);  // -4
```
