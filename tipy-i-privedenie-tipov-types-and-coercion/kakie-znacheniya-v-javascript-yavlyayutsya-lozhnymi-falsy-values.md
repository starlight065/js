# Какие значения в JavaScript являются "ложными" (falsy values)?

Falsy values — это значения, которые приводятся к `false` в булевом контексте.

В JavaScript их немного:

* `false`
* `0`
* `-0`
* `0n`
* `""`
* `null`
* `undefined`
* `NaN`

Например, в `if`, `while` или при использовании `!` все эти значения ведут себя как `false`.

```js
if ("") {
  console.log("не выполнится");
}

if ("0") {
  console.log("выполнится");
}
```

Важно не путать falsy со всеми остальными значениями. Например, `"0"`, `[]` и `{}` — это уже truthy values.
